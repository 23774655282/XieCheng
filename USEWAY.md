# 易宿酒店预订平台 - 使用指南

本文档说明作为**用户**和**商家（商户）**如何使用易宿酒店预订平台。

---

## 一、用户端使用流程

### 1.1 访问入口

- 在手机浏览器或 PC 浏览器打开网站首页：`/`
- 用户端采用响应式布局，手机访问时自动适配移动端体验

### 1.2 搜索酒店

1. **首页查询**
   - 在首页输入**城市**或**关键词**（如酒店名、设施等）
   - 选择**入住日期**和**离店日期**
   - 可点击**快捷标签**（亲子、豪华、停车、近地铁、含早）快速筛选
   - 点击「搜索酒店」进入酒店列表页

2. **列表页筛选**
   - 顶部显示当前查询条件（城市、入住、离店）
   - 可进一步按**星级**、**价格区间**、**标签**等筛选
   - 已选条件会以标签形式展示，可一键清空
   - 列表支持**上滑自动加载**更多酒店

### 1.3 查看酒店详情

1. 在列表页点击任意酒店卡片，进入酒店详情页
2. 可查看：
   - 酒店轮播图
   - 名称、星级、设施、地址
   - 入住/离店日期与间夜信息
   - 房型列表（按价格从低到高排序）

### 1.4 注意事项

- 用户端**仅展示已上线（online）**的酒店
- 处于草稿、审核中、已下线等状态的酒店不会在用户端显示

## 二、商家（商户）使用流程

### 2.1 注册与登录

1. 打开**管理端入口**（首页底部「管理端入口」链接），或直接访问 `/admin/login`
2. 若没有账号，点击「注册」进入 `/admin/register`
3. 注册时选择角色为**商户**
4. 填写用户名、密码后完成注册
5. 使用用户名和密码登录

### 2.2 新增酒店

1. 登录后进入「我的酒店」列表页：`/admin/merchant/hotels`
2. 点击「新增酒店」进入酒店录入页：`/admin/merchant/hotels/new`
3. 填写酒店信息：
   - **必填**：中文名、英文名、地址、星级（1–5）、开业时间
   - **房型**：至少添加一个房型（房型名称、价格）
4. 点击「保存草稿」保存，酒店状态为**草稿（draft）**

### 2.3 编辑酒店

1. 在「我的酒店」列表中点击「编辑」进入编辑页：`/admin/merchant/hotels/:id/edit`
2. 修改信息后：
   - **保存草稿**：继续以草稿形式保存
   - **提交审核**：将酒店提交给管理员审核，状态变为**审核中（pending）**

### 2.4 查看审核状态

在编辑页可查看当前酒店状态：

| 状态 | 说明 |
|------|------|
| 草稿（draft） | 仅保存，未提交审核 |
| 审核中（pending） | 已提交，等待管理员审核 |
| 已通过（approved） | 管理员审核通过，待发布 |
| 已驳回（rejected） | 审核不通过，可查看驳回原因并修改后重新提交 |
| 已上线（online） | 已发布，用户端可见 |
| 已下线（offline） | 已下线，用户端不可见，可申请恢复 |

若为**已驳回**，页面会显示驳回原因，请根据原因修改后重新提交审核。

### 2.5 注意事项

- 商户**只能操作自己名下**的酒店
- 已上线酒店若被商户修改，可能需重新进入审核流程

## 三、管理员使用流程（参考）

> 管理员账号需在注册时选择「管理员」角色，或由系统分配。

### 3.1 登录

- 访问 `/admin/login`，使用管理员账号登录

### 3.2 酒店审核

1. 进入审核列表：`/admin/audit/hotels`
2. 按状态筛选：待审核（pending）、已通过（approved）、已上线（online）、已下线（offline）
3. 点击「审核」进入审核详情页
4. 对待审核酒店执行：
   - **通过**：状态变为已通过（approved）
   - **拒绝**：必须填写驳回原因，状态变为已驳回（rejected）

### 3.3 发布与下线

- **发布**：将已通过（approved）的酒店发布上线，用户端可见
- **下线**：将已上线（online）的酒店下线，用户端不可见
- **恢复**：将已下线（offline）的酒店恢复上线

## 五、常见问题

**Q：用户端搜不到某家酒店？**  
A：可能该酒店尚未上线（online），或处于草稿、审核中、已下线状态。请联系商户或管理员确认。

**Q：商户提交审核后多久能上线？**  
A：需管理员在后台完成审核并发布，具体时间取决于平台运营安排。

**Q：已下线的酒店可以恢复吗？**  
A：可以。管理员可在审核列表中对已下线酒店执行「恢复」操作。
